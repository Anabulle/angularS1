Cours Supplémentaire HttpClient

Introduction à HttpClient

HttpClient est un service fourni par Angular qui permet de réaliser des requêtes HTTP. Il est largement utilisé pour communiquer avec des serveurs et des API externes dans les applications Angular. Ce service rend la gestion des requêtes et des réponses HTTP plus simple et plus intuitive.

Mise en Place de HttpClient

Installation et Configuration

Importation du Module HttpClientModule:

Ajoutez HttpClientModule dans votre module principal (habituellement AppModule).

import { HttpClientModule } from '@angular/common/http';

@NgModule({
  imports: [
    HttpClientModule,
    // autres modules
  ],
})
export class AppModule { }

Injection de HttpClient:

Injectez HttpClient dans les services ou les composants où vous souhaitez l'utiliser.

import { HttpClient } from '@angular/common/http';

@Injectable({
  providedIn: 'root'
})
export class DataService {
  constructor(private http: HttpClient) { }
}

Utilisation de HttpClient

Requêtes de Base

GET: Pour récupérer des données.

this.http.get(url).subscribe(data => {
  // Traitement des données
});

POST: Pour envoyer des données à un serveur.

this.http.post(url, data).subscribe(result => {
  // Traitement du résultat
});

Gestion des Réponses et des Erreurs

Réponses: Traitez les réponses en utilisant .subscribe(). Erreurs: Gérez les erreurs en ajoutant un bloc catchError dans les requêtes.

import { catchError } from 'rxjs/operators';

this.http.get(url).pipe(
  catchError(this.handleError)
).subscribe(
  data => { /* ... */ },
  error => { /* ... */ }
);

Envoyer des Headers et des Paramètres

Headers: Ajoutez des en-têtes à vos requêtes.

const headers = { 'Authorization': 'Bearer votre-token' };
this.http.get(url, { headers }).subscribe(...);

Paramètres de Requête: Ajoutez des paramètres à l'URL de la requête.

const params = new HttpParams().set('userId', '123');
this.http.get(url, { params }).subscribe(...);

Exemple

Créons un service qui interagit avec une API pour gérer des utilisateurs.

@Injectable({
  providedIn: 'root'
})
export class UserService {
  private apiUrl = 'https://monapi.com/utilisateurs';

  constructor(private http: HttpClient) { }

  getUsers() {
    return this.http.get<User[]>(this.apiUrl);
  }

  addUser(user: User) {
    return this.http.post(this.apiUrl, user);
  }
}

interface User {
  id: number;
  nom: string;
}
